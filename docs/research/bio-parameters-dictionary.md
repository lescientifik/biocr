## Notes sur les plages plausibles

Les plages indiquées sont **physiologiquement possibles**, pas les normes de référence. Elles servent à détecter les erreurs OCR évidentes :
- Une glycémie à 150 g/L est clairement une erreur (plausible max ~8 g/L)
- Une glycémie à 1.50 g/L est plausible même si anormalement élevée
- Les plages tiennent compte des situations pathologiques extrêmes
